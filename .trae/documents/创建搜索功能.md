# 创建搜索功能

## 1. 创建前端搜索页面 Search.vue
- 参考首页的课程列表样式
- 支持搜索关键词输入
- 显示搜索结果
- 支持加载更多和下拉刷新

## 2. 添加搜索路由
- 在 router/index.js 中添加 `/search` 路由
- 设置页面标题为"搜索"

## 3. 添加前端搜索 API
- 在 api/course.js 中添加 searchCourses 方法
- 调用后端搜索接口

## 4. 添加后端搜索接口
- 在 backend/controllers/course.js 中添加 searchCourses 方法
- 支持按关键词搜索课程
- 支持分页

## 5. 添加后端搜索路由
- 在 backend/routes/course.js 中添加搜索路由
- 添加参数验证

## 预期效果
- 用户可以通过搜索框搜索课程
- 搜索结果页面正常显示
- 支持分页加载更多
