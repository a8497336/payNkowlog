- 调整轮播图高度和圆角
- 优化指示器样式
- 添加图片加载占位效果

## 2. 创建登录页面
- 创建 `Login.vue` 页面
- 包含用户名和密码输入框
- 添加登录按钮和表单验证
- 美化登录页面样式

## 3. 添加后端登录接口
- 在后端添加用户名密码登录接口
- 验证用户名密码
- 返回 JWT token

## 4. 更新前端登录 API
- 添加用户名密码登录方法
- 更新登录状态管理

## 5. 更新路由守卫
- 修改路由守卫逻辑
- 未登录时跳转到登录页面而不是首页
- 登录成功后跳转到原访问页面

## 6. 更新路由配置
- 添加登录页面路由
- 设置登录页面为公开路由

## 预期效果
- 轮播图显示美观，尺寸合适
- 未登录访问需要认证的页面时跳转到登录页
